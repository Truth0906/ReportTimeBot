# ReportTimeBot

###### 這是專為 PTT 爬蟲開發者所開發的函式庫 [PTTCrawlerLibrary](https://github.com/Truth0906/PTTCrawlerLibrary) 的範例程式
###### 會根據報時檔案，時間到就 PO 文
###### 預先載入下一分鐘的報時檔案 絕不失誤

如何使用
-------------------
###### 如果你想要在 12:32 準點 PO 文，那就創一個 1232.txt
###### 第一行為標題，第二行之後為文章內文
###### {TIME} 會自動代換為當時的時間
